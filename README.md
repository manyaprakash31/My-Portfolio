# Personal Portfolio Website

A modern, responsive personal portfolio website built using Flask.
It showcases my projects in AI, Full-Stack Development, and Data Science,
along with role-based resumes and direct contact options.

## Features
- Modern dark UI with custom design
- AI, Full-Stack & Data Science project showcase
- Role-based resumes (Developer & Data Science)
- Direct email, LinkedIn, and GitHub links
- Flask-based and easy to deploy

## Tech Stack
- Python, Flask
- HTML, CSS
- Font Awesome

## Run Locally
```bash
pip install flask
python app.py
